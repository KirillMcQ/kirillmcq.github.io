# Write documents!

This is a site where you can write documents, like google docs and word. I am currently implementing this on github pages to see how it works and then I will publish it as a desktop app with electron. The installer will be here once I release the first version!

Changelog:
V0.0 The first version. Just the bare features of underline, highlight, change text color, and italics. More to come soon!

V0.1 Added Use.js for readability. 

V0.2 Added a bullet point button to add a bullet point when clicked.

V.0.3 Added a font size changer.
